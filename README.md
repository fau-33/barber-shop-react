# Barber Shop React

Este é um projeto de uma página web moderna e responsiva para uma barbearia, desenvolvido utilizando as mais recentes tecnologias e práticas de desenvolvimento web.

## Tecnologias Utilizadas

- **Next.js**: Framework React para renderização do lado do servidor (SSR) e geração de sites estáticos (SSG), proporcionando performance e SEO aprimorados.
- **React**: Biblioteca JavaScript para construção de interfaces de usuário reativas e componentizadas.
- **React Icons**: Biblioteca que oferece uma vasta coleção de ícones (como FiSun e FiMoon) de forma fácil e performática.
- **CSS Modules**: Para uma estilização modular, garantindo escopo local para os estilos dos componentes e evitando conflitos de nomes.
- **Next/Image**: Componente otimizado do Next.js para carregamento e renderização eficiente de imagens, melhorando a performance e a experiência do usuário.
- **Next/Font**: Recurso do Next.js para carregamento otimizado de fontes, garantindo melhor desempenho e conformidade visual.

## Instalação e Execução

Para rodar este projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/seu-usuario/barber-shop-react.git
    cd barber-shop-react
    ```

2.  **Instale as dependências:**

    ```bash
    npm install
    # ou yarn install
    ```

3.  **Inicie o servidor de desenvolvimento:**

    ```bash
    npm run dev
    # ou yarn dev
    ```

4.  Abra seu navegador e acesse `http://localhost:3000`
